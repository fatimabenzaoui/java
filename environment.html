CREATE A SPRING BOOT PROJECT
Ctrl + Shift + P
Taper spring
Sélectionner Spring Initializer : Create a Maven Project
Sélectionner Version de Spring Boot : choisir la dernière version stable (éviter les versions SNAPSHOT)
Sélectionner le langage du projet : java
Préciser le Group (package name) : com.app
Préciser l'Artifact (nom du projet): myFirstApp
Sélectionner le type du Packaging : Jar
Sélectionner la version de Java : 17
Cocher les dépendances : 
    - Spring Web
    - Spring Boot DevTools

// Lancer l'application ds l'IDE
Se rendre sur MyappApplication.java / Cliquer sur Run
Se rendre sur : http://localhost:8080 (// affiche Whitelabel Error Page)

// Lancer l'application ds l'invite de cmd
// Créer le myapp.jar
mvnw package ou ./mvnw package
cd target
java -jar myapp-0.0.1-SNAPSHOT.jar
Ctrl + C (pour fermer le serveur)
ou
cd ..
mvnw spring-boot:run ou ./mvnw spring-boot:run

// Changer le port si besoin ds application.properties
server.port = 8081
